# BTC Price Next.js App

This project is a minimal Next.js application that displays the current price of Bitcoin in USD using the [CoinGecko API](https://www.coingecko.com/en/api).

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) to see the page.
